[Operator specifiation for ts_std]
1. Type signature
ts_std: Signal<Float> (signal: Signal<Float>, period: Int)
2. Arguments
signal: Input signal.
period: Lookback window. Integer greater than 1.
3. Semantic Definition 
Measures the historical volatility of signal within the window.
The result is often used for normalization.
4. Example
event = ts_std(cs_rank(signal=volume), period=5)
result = multiply(x=ts_rank(signal=price, period=5), y=event)
