[Operator specifiation for less]
1. Type signature
less: Signal<Float> (signal: Either(Signal<Float>, Float), thres: Either(Signal<Float>, Float))
2. Arguments
signal: Input signal.
thres: Input signal, or a number for threshold
3. Semantic Definition
Outputs 1.0 if the input signal is less than the threshold 
4. Example
is_loss = less(signal=return, thres=0.0)
