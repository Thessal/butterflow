[Operator specifiation for divide]
1. Type signature
divide: Signal<Float> (dividend: Either(Signal<Float>, Float), divisor: Either(Signal<Float>, Float))
2. Arguments
dividend: Input signal.
divisor: Nonzero input signal.
3. Semantic Definition
Divides dividend by divisor.
Useful for calculating financial ratios.
4. Example
returns = subtract(x=divide(dividend=close, divisor=open), y=1.0)
eps = divide(divdend=ebit, divisor=sharesout)
