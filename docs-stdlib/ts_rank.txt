[Operator specifiation for ts_rank]
1. Type signature
ts_rank: Signal<Float> (signal: Signal<Float>, period: Int)
2. Arguments
signal: Input signal.
period: Lookback window. Integer greater than 1.
3. Semantic Definition 
Scales the signal from 0 to 1, by comparing current data to the historical range.
4. Example
price_trend = ts_rank(signal=close, period=200)
